# 🎮 2D Game Portfolio

Welcome to my **2D Game Portfolio**! 🌟 This interactive game brings my developer portfolio to life in a creative and fun way, showcasing my skills and experiences through an engaging 2D world that represents my workspace. 🎮

## 🚀 Features

- 🧑‍💻 **Explore my interactive 2D environment**, which represents my developer workspace—everything from my desk to the code I’ve written.
- 🕹️ **Control the player** with arrow keys or mobile touch controls to move around and discover different areas of my portfolio.
- 💬 **Learn about my skills, projects, and experiences** through dialogue boxes that pop up as you interact with the world.
- 🔗 **Click on objects** to access external links like my resume, GitHub profile, and other important parts of my online presence.
- 📱 **Enjoy a responsive design**, ensuring the portfolio looks great and works smoothly on both desktop and mobile devices!

## 🛠️ Technologies Used

- **JavaScript**: Powers the interactivity and functionality of my portfolio, making it dynamic.
- **Kaboom.js**: The game engine that brings my creative ideas to life in this 2D portfolio world. 🎮
- **Vite**: Speeds up development and provides a seamless build tool and server.
- **HTML5 & CSS3**: Used to structure and style my portfolio, giving it an aesthetic and user-friendly look.
- **Tiled**: Designed my game map, bringing the physical space of my developer environment to the digital world. 🗺️

## 🏃‍♂️ How to Run

1. **Clone the repository** to bring the project to your local machine:
    ```bash
    https://github.com/Ma-n-a-v/my-2d-game-portfolio.git
    ```

2. **Install the dependencies** to get everything set up:
    ```bash
    npm install
    ```

3. **Run the development server** to start the game and explore my portfolio:
    ```bash
    npm run dev
    ```

4. Open the provided localhost URL in your browser to dive into my world! 🎉

## 📂 Project Structure

- **`src/`**: Contains the core JavaScript files, which power the game mechanics and interactive features of my portfolio.
- **`public/`**: Stores the assets that make the game visually engaging—sprites, fonts, sounds, and more.
- **`index.html`**: The entry point to my interactive world!
- **`vite.config.js`**: Configures Vite to ensure a smooth development process and fast builds. ⚙️
